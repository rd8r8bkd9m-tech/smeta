# Смета - Приложение для составления и редактирования смет

Веб-приложение для замерщиков и специалистов по составлению смет. Позволяет создавать, редактировать и управлять сметами с автоматическим расчетом стоимости.

## Возможности

- ✅ Создание и редактирование смет
- ✅ Добавление неограниченного количества позиций
- ✅ Автоматический расчет итоговой стоимости
- ✅ Поддержка различных единиц измерения (шт, м, м², м³, кг, т, час, день)
- ✅ Сохранение смет в браузере (localStorage)
- ✅ Экспорт в PDF/печать
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Полностью на русском языке

## Использование

### Способ 1: Открыть напрямую в браузере

Просто откройте файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Safari, Edge).

### Способ 2: Запустить локальный сервер

```bash
# С помощью Python 3
python3 -m http.server 8080

# С помощью Node.js (если установлен http-server)
npx http-server -p 8080

# Затем откройте http://localhost:8080 в браузере
```

## Как пользоваться

1. **Создать новую смету**: Нажмите кнопку "Создать новую смету"
2. **Заполнить информацию**: Введите название, дату, клиента и проект
3. **Добавить позиции**: Нажмите "Добавить позицию" и заполните:
   - Наименование работ/материалов
   - Количество
   - Единица измерения
   - Цена за единицу
4. **Автоматический расчет**: Сумма по каждой позиции и итого рассчитываются автоматически
5. **Сохранить**: Нажмите кнопку "Сохранить" для сохранения сметы
6. **Печать/Экспорт**: Используйте кнопку "Печать" для создания PDF
